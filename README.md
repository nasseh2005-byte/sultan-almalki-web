# Sultan Al-Maliki Legal Website

Next.js App Router website for **Sultan Mohammed Al-Maliki Attorneys & Legal Consultants**.

## What Is Included

- Arabic and English public routes: `/ar` and `/en`.
- Premium legal-tech visual direction with dark and day themes.
- Elegant click sound on primary interactions using the Web Audio API.
- Partner logo grid using approved public assets.
- Circular KPI statistics, bar charts, donut chart, and accessible data tables.
- A compact `/admin` snapshot page for internal analytics framing.
- Approved public aggregate metrics in `src/lib/data.ts`.

## Local Development

```bash
npm install
npm run dev
```

Open:

```text
http://127.0.0.1:3000/ar
http://127.0.0.1:3000/en
http://127.0.0.1:3000/admin
```

## Checks

```bash
npm run lint
npm run typecheck
npm run build
```

## Data Privacy

Do not commit private Excel files, case numbers, payment numbers, Google Drive links, raw documents, opponent details, or credentials.

Only aggregated and approved public snapshots should be committed after explicit publication approval. Private source files and unapproved internal snapshots belong in secure storage outside GitHub.
